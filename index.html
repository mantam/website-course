HTML/CSS/Javascript Introductory Course
In this course, you will learn the skills needed to develop a multi-page website. By the end of this course, which includes 20 (45 minutes) one-on-one classes, you will finish building a website using HTML5, CSS3, and Javascript. Here is a functional example of the website. Homework is given at the end of each class and must be done by the beginning of next class. Although we are keeping the design of the website the same, you will be creating all of the images and providing all of the content. In addition to javascript, you will also learn "mustache.js", a popular front-end templating engine.
Syllabus
  1. Overview of website pages (categories -> items -> detail)
  2. Image creating/editing with Paint.NET
  3. Hypertext Markup Language (HTML) basics
  4. Cascading Style Sheets (CSS) basics
  5. Catch up class 1
  6. HTML / CSS II
  7. Page Header / Footer
  8. CSS responsive styling with media queries
  9. Javascript basics
  10. Catch up class 2
  11. Javascript objects (JSON) and statements
  12. Javascript HTML interactions
  13. Javascript data / url parameters
  14. Search (CSS transition / HTML datalist for input)
  15. Catch up class 3
  16. Mustache.js templating engine (basic syntax)
  17. JSON data for website
  18. Mustache.js templates - footer & header
  19. Mustache.js template - detail page
  20. Catch up class 4
Catch up classes are used for Q&A, more 1-on-1 examples, resolve issues with student's web pages and review all homework assignments.
